# 🎓 Aluno+

> Plataforma de acompanhamento estudantil para organização, monitoramento e melhoria do desempenho académico dos alunos.

---

## 📌 Descrição

O **Aluno+** é uma plataforma web desenvolvida para ajudar estudantes a acompanhar o seu progresso académico de forma simples e organizada.

A aplicação permite gerir disciplinas, tarefas, notas e metas de estudo, oferecendo uma visão clara do desempenho do aluno ao longo do tempo.

---

## 🚀 Tecnologias Utilizadas

- ⚛️ React
- 🎨 TailwindCSS
- ⚡ JavaScript (ES6+)
- ⚡ Vite
- 🌐 React Router

---

## ✨ Funcionalidades

- 📚 Gestão de disciplinas
- 📝 Registo de tarefas e trabalhos
- 📊 Acompanhamento de notas
- 🎯 Definição de metas de estudo
- 📅 Calendário académico
- 📱 Interface responsiva (mobile e desktop)
- 📈 Visão geral do desempenho do aluno
